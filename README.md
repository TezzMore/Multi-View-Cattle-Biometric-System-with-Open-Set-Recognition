# Multi-View-Cattle-Biometric-System-with-Open-Set-Recognition
Deep Metric Learning for Livestock Identification
◦ Deep Learning Architecture: Developed ResNet-Transformer pipeline with pretrained ResNet50 backbone and
multi-head attention, trained using ArcFace loss with angular margin for enhanced inter-class separation in embedding
space.
◦ Multi-Modal Fusion: Engineered paired dataset framework processing face and muzzle images with gallery-based
verification using weighted cosine similarity fusion and validation-driven threshold estimation.
◦ Open-Set Recognition: Implemented conservative decision system with accept/reject/low-confidence classification,
prioritizing low false acceptance through adaptive thresholds for unknown cattle detection.
◦ Technologies: Python, PyTorch, ResNet, Transformer, ArcFace, Metric Learning, Transfer Learning.
